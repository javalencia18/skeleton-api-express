## Get started developing

### Install project dependencies

```bash
npm install
```

### Init husky

```bash
npm run prepare
```

## Building

```bash
npm run build
```

## Testing

### Jest with Testing Library

```bash
npm run test
```

## Linting

Run the linter

```bash
npm run lint
```

Fix lint issues automatically

```bash
npm run lint:fix
```
